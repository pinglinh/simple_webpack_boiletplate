## Simple React Boilderplate
> A ready-to-use boilerplate for [facebook/react](https://github.com/facebook/react/).

### Versions
| Package | Version |
|---------|---------|
| Babel   | 7.x.x   |
| Webpack | 4.x.x   |
| React   | 16.x.x  |

### Installation
1. `npm install`
2. `npm start`
